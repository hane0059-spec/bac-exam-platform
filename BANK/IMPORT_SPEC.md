# معيار بنك الأسئلة الخارجي — صيغة الاستيراد إلى المنصّة

> هذا المرجع **مُلزِم**. أي ملفّ JSON يلتزم به يدخل بنك المنصّة **دون أي تعديل**.
> الخطوة اليدوية الوحيدة المسموح بها بعد الاستيراد: **رفع المدرّس لصورة سؤال الرسم** (DIAGRAM_LABEL).

---

## 1) القاعدة الذهبية
- أخرِج **JSON صالحاً** بترميز **UTF-8** (عربيّة مباشرة، لا ترميز `\uXXXX`).
- لا تخترع حقولاً جديدة ولا تغيّر أسماء الحقول. التزم بالأسماء الإنجليزية أدناه **حرفياً**.
- النصوص العربية تُكتب كما هي داخل القيم النصّية.
- المنصّة **لا تستورد صوراً**؛ أي شكل/منحنى يُوصَف نصّاً (انظر `*_description`).

---

## 2) البنية العامة للملفّ
```json
{
  "metadata": {
    "subject": "علم الأحياء",
    "grade": "الصف الثالث الثانوي - العلمي",
    "version": "2026"
  },
  "questions": [ { ... }, { ... } ]
}
```
- `metadata.subject`: نصّ **للعرض فقط** (المادة الفعلية يختارها المدير/المدرّس عند الرفع).
- `questions`: مصفوفة الأسئلة (إلزامية). الملفّ بلا `questions` يُرفَض كلّه.

---

## 3) الحقول المشتركة لكل سؤال
| الحقل | إلزاميّ؟ | القيمة |
|---|---|---|
| `id` | مُستحسَن | معرّف نصّيّ فريد (يظهر في تقرير الأخطاء، مثل `"BIO-001"`). |
| `type` | **نعم** | نوع السؤال (انظر القسم 6). |
| `concept_id` | اختياري | معرّف الدرس في مصدرك (مثل `"C5.2"`)؛ يُحفظ كوسم `مرجع:C5.2`. |
| `level` | اختياري | `"easy"` / `"medium"` / `"hard"` / `"expert"` ← الصعوبة (الافتراضي `medium`). |

### اشتقاق العلامة (points)
تُؤخذ علامة السؤال تلقائياً بهذا الترتيب: `total_marks` ← `marks` ← مجموع `scoring_rubric[].marks` ← وإلّا **1**. (محدودة بين 1 و999.99.)

---

## 4) كل سؤال يجب أن «يُصحَّح» — أين توضع الإجابة الصحيحة
- الأنواع الآلية (اختيار/صح-خطأ/ترتيب/مطابقة/ملء فراغات/توسيم رسم): الإجابة الصحيحة **إلزامية** في الحقل المخصّص لكلٍّ.
- الأنواع المقالية: ضع **نموذج الإجابة** في أحد الحقول المعتمَدة (يظهر للمدرّس عند التصحيح اليدويّ).

---

## 5) خريطة الأنواع (ماذا يصبح كل نوع داخل المنصّة)
| اكتب `type` = | يصبح في المنصّة | التصحيح |
|---|---|---|
| `multiple_choice` | اختيار من متعدّد | آليّ |
| `true_false` | صح/خطأ | آليّ |
| `ORDER` | ترتيب | آليّ |
| `MATCH` | مطابقة | آليّ |
| `CONCEPT_MAP` | ملء فراغات | آليّ |
| `DIAGRAM_LABEL` | توسيم رسم (الصورة تُرفع لاحقاً) | آليّ |
| `ESSAY` | مقاليّ | يدويّ |

**أسماء بديلة مقبولة** (تُعامَل كنظيرتها):
- بديل `multiple_choice`: `MCQ_TEXT`, `analysis`, `image_interpretation`, `MCQ_GRAPH`, `MCQ_IMAGE`, `MCQ_DIAGRAM`, `MCQ_PEDIGREE`.
- بديل `true_false`: `TF`.
- بديل `DIAGRAM_LABEL`: `LABEL`.
- بديل `ESSAY`: `EXPLAIN`, `COMPARE`, `GENETICS`, `CRITICAL`, `RESULT`, `LOCATE`, `FUNCTION`, `PEDIGREE` (وأي نوع غير معروف → مقاليّ).

> **للتبسيط استعمل الأسماء القانونية في العمود الأول.**

---

## 6) مواصفة كل نوع + مثال

### 6.1 اختيار من متعدّد — `multiple_choice`
الحقول: `stem` (أو `text`)، `options` (مصفوفة `{key, text}`)، `answer` (مفتاح الخيار الصحيح)، `explanation` (اختياري).
- عدد الخيارات: **2 إلى 6**، وإجابة صحيحة **واحدة** تطابق أحد المفاتيح.
- سؤال مصوَّر؟ صف الشكل في `graph_description` / `image_description` / `diagram_description` / `pedigree_description` (يُدمَج في نصّ السؤال).
```json
{
  "id": "Q1", "type": "multiple_choice", "concept_id": "C2.1", "level": "easy",
  "text": "أي البنى ليست جزءاً من جذع الدماغ؟",
  "options": [
    {"key": "A", "text": "البصلة السيسائية"},
    {"key": "B", "text": "الحدبة الحلقية"},
    {"key": "C", "text": "المخيخ"}
  ],
  "answer": "C",
  "explanation": "المخيخ جزء مستقلّ متّصل بجذع الدماغ وليس منه."
}
```

### 6.2 صح/خطأ — `true_false`
الحقول: `stem` (أو `text`)، `answer`، و`explanation` (أو `correction`) اختياري.
- صيغة مبسّطة: `answer` = `"صحيح"` أو `"خطأ"` (يكفي أن تبدأ بـ«صح» لتُعدّ صحيحة).
```json
{
  "id": "Q2", "type": "true_false", "concept_id": "C2.2", "level": "medium",
  "stem": "ينشأ الجهاز العصبي من الوريقة الجنينية الخارجية.",
  "answer": "صحيح",
  "correction": "نعم، تزداد ثخانة الوريقة الخارجية لتشكّل اللويحة العصبية."
}
```

### 6.3 ترتيب — `ORDER`
الحقول: `stem`، `shuffled_steps` (مصفوفة نصوص بأي ترتيب)، `correct_order` (مصفوفة أرقام **تبدأ من 1** تشير إلى مواضع الخطوات بالترتيب الصحيح).
- عدد الخطوات: **2 إلى 8**؛ طول `correct_order` = طول `shuffled_steps`.
```json
{
  "id": "Q3", "type": "ORDER", "level": "hard",
  "stem": "رتّب مراحل تشكّل الأنبوب العصبي:",
  "shuffled_steps": ["تلتحم الطيتان وتتحوّل إلى أنبوب", "تتشكّل لويحة عصبية", "تظهر طيتان وميزابة"],
  "correct_order": [2, 3, 1]
}
```
> `correct_order` يعني: الخطوة رقم 2 أوّلاً، ثمّ 3، ثمّ 1.

### 6.4 مطابقة — `MATCH`
الحقول: `stem`، `column_A` (يسار، كلٌّ بصيغة `"رقم- نصّ"`)، `column_B` (يمين، كلٌّ بصيغة `"حرف- نصّ"`)، `answer_key` (كائن يربط **رقم A → حرف B**).
- عدد الأزواج: **2 إلى 8**؛ كل مفتاح يجب أن يُحلَّ.
- البادئات `1- ` و`أ- ` تُزال تلقائياً، وحرف العمود B = أوّل محرف من النصّ.
```json
{
  "id": "Q4", "type": "MATCH", "level": "medium",
  "stem": "طابق الكائن بوصف جهازه العصبي:",
  "column_A": ["1- الهيدرا", "2- دودة الأرض"],
  "column_B": ["أ- حبل عصبي بطني وعقد", "ب- شبكة عصبية في كل الاتجاهات"],
  "answer_key": {"1": "ب", "2": "أ"}
}
```

### 6.5 ملء فراغات — `CONCEPT_MAP`
الحقول: `stem`، `structure` (نصّ فيه فراغات بالشكل `[رقم: تلميح]`)، `empty_nodes` (كائن: لكل رقم إجابته).
- عدد الفراغات: **1 إلى 8**؛ لكل فراغ في `structure` إجابةٌ في `empty_nodes`.
- مرادفات الفراغ الواحد تُفصَل بـ`|` داخل قيمة `empty_nodes`.
```json
{
  "id": "Q5", "type": "CONCEPT_MAP", "level": "medium",
  "stem": "أكمل مخطّط الجهاز العصبي:",
  "structure": "الجهاز العصبي يتكوّن من [1: قسم مركزي] و[2: أعصاب].",
  "empty_nodes": {"1": "الجهاز العصبي المركزي", "2": "الأعصاب|الجهاز العصبي المحيطي"}
}
```

### 6.6 توسيم رسم — `DIAGRAM_LABEL`  ⟵ الوحيد الذي يحتاج خطوةً يدوية
الحقول: `stem` (أو `text`)، `correct_labels` (كائن: رقم السهم → التسمية الصحيحة)، و`image_description` (اختياري، وصفٌ مؤقّت)، و`explanation` (اختياري).
- عدد التسميات: **1 إلى 12**؛ مرادفات كل تسمية تُفصَل بـ`|`.
- **الخطوة اليدوية**: بعد الاستيراد يفتح المدرّس السؤال ويرفع **الصورة المرقّمة**؛ يجب أن تطابق أرقام الأسهم في الصورة مفاتيح `correct_labels`.
```json
{
  "id": "Q6", "type": "DIAGRAM_LABEL", "level": "medium",
  "stem": "لاحظ مقطع كرة العين واكتب مسمّى كل رقم.",
  "image_description": "مقطع كرة العين بأسهم مرقّمة 1-3",
  "correct_labels": {"1": "القرنية", "2": "القزحية", "3": "الشبكية|الشبكِيّة"}
}
```

### 6.7 مقاليّ — `ESSAY` (وأسماؤه البديلة)
نصّ السؤال يُبنى من أوّل متوفّر: `stem` ← `case` ← `phenomenon` ← `text` ← (`topic_A` + `topic_B` + `comparison_aspects`).
ضع **نموذج الإجابة** في أيٍّ من هذه (تظهر للمدرّس): `explanation` أو `key_points` (مصفوفة) أو `scoring_rubric` (مصفوفة `{point, marks}`) أو `items` (مصفوفة `{term, correct_location}` أو `{term, one_function}`) أو `answer_table` (`{المحور: [قيمةA, قيمةB]}` مع `topic_A`/`topic_B`) أو `full_solution` (كائن) أو `answer_key` (نصّ أو كائن).
```json
{
  "id": "Q7", "type": "ESSAY", "level": "hard",
  "stem": "اشرح آلية انتقال السيالة عبر المشبك الكيميائي.",
  "total_marks": 12,
  "key_points": [
    "وصول كمون العمل إلى الزر النهائي",
    "دخول الكالسيوم وتحرّر الناقل في الفالق",
    "ارتباط الناقل بمستقبلات الغشاء بعد المشبكي"
  ]
}
```

## 7) حدود التحقّق (تجاوزها يَستبعد السؤال وحده، لا الملفّ)
- اختيار: 2–6 خيارات، إجابة واحدة تطابق مفتاحاً.
- صح/خطأ: إجابة واضحة (صحيح/خطأ).
- ترتيب: 2–8 خطوات، `correct_order` مطابق الطول وصحيح المؤشّرات.
- مطابقة: 2–8 أزواج، كل مفتاح يُحلّ.
- ملء فراغات: 1–8 فراغات، لكل فراغ إجابة.
- توسيم رسم: 1–12 تسمية مرقّمة.
- المقاليّ: نصّ سؤال غير فارغ.

السؤال غير المطابق يظهر في «المرفوضة» بسبب واضح، وتُستورَد البقيّة.

---

## 8) أخطاء شائعة تُرفَض — تجنّبها
- `answer` لا يطابق أيّ `key` في الاختيار.
- `correct_order` أرقامه تبدأ من 0 (يجب أن تبدأ من **1**).
- `answer_key` في المطابقة يشير إلى حرف غير موجود في `column_B`.
- فراغ في `structure` بلا مفتاح مقابل في `empty_nodes`.
- خلط الأنواع: وضع `options` لسؤال ترتيب، أو `shuffled_steps` لسؤال اختيار.
- إخراج Markdown أو تعليقات داخل JSON (JSON صِرف فقط).

---

## 9) تعليمات جاهزة للّصق إلى الذكاء الاصطناعي
> أنشئ بنك أسئلة بصيغة JSON متوافقة **حرفياً** مع «معيار بنك الأسئلة الخارجي» التالي.
> - أخرِج JSON صِرفاً (بلا Markdown ولا شرح خارجه) بترميز UTF-8 وعربيّة مباشرة.
> - الجذر: `{ "metadata": { "subject": "<المادة>" }, "questions": [ ... ] }`.
> - لكل سؤال: `id` فريد، و`type` من القائمة القانونية فقط
>   (`multiple_choice` / `true_false` / `ORDER` / `MATCH` / `CONCEPT_MAP` / `DIAGRAM_LABEL` / `ESSAY`)،
>   و`level` (easy/medium/hard)، والحقول الإلزامية لذلك النوع كما في المعيار.
> - ضع الإجابة الصحيحة في حقلها المخصّص لكل نوع آليّ، ونموذج إجابة لكل مقاليّ.
> - صف أي شكل/منحنى نصّاً في `*_description`؛ ولأسئلة `DIAGRAM_LABEL` ضع `correct_labels` المرقّمة (الصورة يرفعها المدرّس لاحقاً).
> - لا تستعمل حقولاً خارج المعيار. النطاق: <حدّد الدرس/الصفحات>.

---

## 10) مثال ملفّ كامل صغير (جاهز للاستيراد)
```json
{
  "metadata": { "subject": "علم الأحياء", "version": "2026" },
  "questions": [
    {
      "id": "B1", "type": "multiple_choice", "level": "easy",
      "text": "الغذاء الرئيس لخلايا الدماغ هو:",
      "options": [
        {"key": "A", "text": "البروتين"},
        {"key": "B", "text": "الغلوكوز"},
        {"key": "C", "text": "الدهون"}
      ],
      "answer": "B"
    },
    {
      "id": "B2", "type": "true_false", "level": "easy",
      "stem": "النخاع الشوكي يتشكّل من القسم المتبقّي من الأنبوب العصبي.",
      "answer": "صحيح"
    },
    {
      "id": "B3", "type": "DIAGRAM_LABEL", "level": "medium",
      "stem": "اكتب مسمّى كل رقم في مقطع العين.",
      "image_description": "مقطع كرة العين بأسهم 1-3",
      "correct_labels": {"1": "القرنية", "2": "العدسة", "3": "الشبكية"}
    }
  ]
}
```
