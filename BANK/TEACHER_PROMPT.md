# تعليمات المدرّس: توليد بنك أسئلة جاهز للاستيراد

## كيف تستعمله (٤ خطوات)
1. انسخ **النصّ الكامل** بين الخطّين أدناه وألصقه للذكاء الاصطناعي عندك.
2. في آخره ألصق نصّ الدرس (أو ارفق صفحات الكتاب) وحدّد المادة والنطاق.
3. احفظ ناتج الـ JSON في ملفّ بامتداد `.json`.
4. في المنصّة: **بنك الأسئلة ← استيراد من ملفّ ← اختر المادة ← ارفع الملفّ ← عاين ← أكّد**.
   - أسئلة الرسم: افتح السؤال بعد الاستيراد وارفع **الصورة المرقّمة**.

---

أنت مولّد بنك أسئلة. حوّل المحتوى الذي سأعطيك إلى ملفّ **JSON** بصيغة دقيقة لمنصّة اختبارات، بحيث يُستورَد **دون أي تعديل**.

**قواعد الإخراج (إلزامية):**
- أخرِج **JSON صالحاً فقط** — لا أي كلام أو شرح أو Markdown قبله أو بعده.
- الترميز UTF-8 والعربية مكتوبة مباشرةً (لا ترميز مثل `أ`).
- لا تُضِف حقولاً غير المذكورة، ولا تُغيّر أسماء الحقول (تبقى بالإنجليزية حرفياً).
- لكل سؤال إجابته الصحيحة في حقلها المخصّص (للمقاليّ: نموذج إجابة).

**شكل الملفّ:**
```
{
  "metadata": { "subject": "اسم المادة" },
  "questions": [ ... ]
}
```

**حقول مشتركة لكل سؤال:**
- `"id"`: معرّف فريد نصّي (مثل `"Q1"`).
- `"type"`: نوع السؤال من القائمة أدناه **حرفياً**.
- `"level"`: `"easy"` أو `"medium"` أو `"hard"`.
- اختياري `"concept_id"`: اسم الدرس أو الوحدة.
- اختياري `"total_marks"`: علامة السؤال (الافتراضي 1).

**الأنواع وحقولها:**

1) **اختيار من متعدّد** — `"type": "multiple_choice"`
   `"text"` (نص السؤال)، `"options"`: مصفوفة `{"key":"A","text":"..."}` (من 2 إلى 6)، `"answer"` (مفتاح الصحيح)، واختياري `"explanation"`.
   مثال: `{"id":"Q1","type":"multiple_choice","level":"easy","text":"...؟","options":[{"key":"A","text":"..."},{"key":"B","text":"..."}],"answer":"B"}`

2) **صح/خطأ** — `"type": "true_false"`
   `"stem"` (العبارة)، `"answer"`: `"صحيح"` أو `"خطأ"`، واختياري `"correction"`.

3) **ترتيب** — `"type": "ORDER"`
   `"stem"`، `"shuffled_steps"`: مصفوفة خطوات بأي ترتيب (2 إلى 8)، `"correct_order"`: أرقام **تبدأ من 1** تدلّ على الترتيب الصحيح (نفس عدد الخطوات).
   مثال: `"shuffled_steps":["ب","ج","أ"]` مع `"correct_order":[3,1,2]` يعني الترتيب الصحيح: أ ثمّ ب ثمّ ج.

4) **مطابقة** — `"type": "MATCH"`
   `"stem"`، `"column_A"`: `["1- ...","2- ..."]`، `"column_B"`: `["أ- ...","ب- ..."]`، `"answer_key"`: `{"1":"ب","2":"أ"}` (يربط رقم A بحرف B، من 2 إلى 8 أزواج).

5) **ملء فراغات** — `"type": "CONCEPT_MAP"`
   `"stem"`، `"structure"`: نصّ فيه فراغات بالشكل `[1: تلميح]` و`[2: تلميح]` (من 1 إلى 8)، `"empty_nodes"`: `{"1":"الإجابة","2":"الإجابة"}`. (المرادفات داخل الإجابة تُفصَل بـ`|`.)

6) **توسيم رسم** (السؤال الذي يحتاج صورة) — `"type": "DIAGRAM_LABEL"`
   `"stem"` (مثل: اكتب مسمّى كل رقم)، `"correct_labels"`: `{"1":"...","2":"..."}` (رقم السهم ← التسمية الصحيحة، من 1 إلى 12)، واختياري `"image_description"`.
   ملاحظة: الصورة المرقّمة يرفعها المدرّس داخل المنصّة لاحقاً؛ اجعل الأرقام منطقية ومتسلسلة.

7) **مقاليّ** — `"type": "ESSAY"`
   `"stem"` (نص السؤال)، `"key_points"`: مصفوفة عناصر الإجابة النموذجية، واختياري `"total_marks"`.

**حدود مهمّة (لئلّا يُرفَض السؤال):**
- الاختيار: 2–6 خيارات وإجابة واحدة تطابق أحد المفاتيح.
- الترتيب: الأرقام تبدأ من 1 وعددها = عدد الخطوات.
- المطابقة: كل رقم في `answer_key` يقابل حرفاً موجوداً في `column_B`.
- ملء الفراغات: لكل `[رقم: …]` في `structure` إجابةٌ في `empty_nodes`.
- لا تخلط حقول نوعٍ مع نوع آخر، ولا تكتب أرقاماً عربية في المفاتيح.

**الآن أخرِج ملفّ JSON كاملاً للمحتوى التالي:**
المادة: «اكتب اسم المادة هنا». النطاق: «اكتب الدرس أو الصفحات». المحتوى:
«ألصق نصّ الدرس هنا، أو ارفق صفحات الكتاب».

---

> بعد أن يعطيك الذكاء الاصطناعي الـ JSON: احفظه باسم مثل `bank.json` وارفعه من صفحة الاستيراد. إن ظهرت أسئلة في خانة «المرفوضة» فالسبب يظهر بجانب كل سؤال (غالباً مخالفة لأحد الحدود أعلاه) — صحّحها وأعد الرفع، أو استورد البقيّة.
